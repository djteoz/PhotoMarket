# Проект: Агрегатор Фотостудий (Photo Studio Marketplace)

## Цель

Создать полноценный Fullstack-проект для портфолио и последующей продажи. Агрегатор должен позволять владельцам студий размещать свои залы, а фотографам — искать и бронировать их.

## Технический стек (Modern Fullstack)

- **Frontend**: Next.js (App Router), React
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Kit**: Shadcn/UI (современный, доступный, легко кастомизируемый)
- **Database**: PostgreSQL (надежная реляционная БД)
- **ORM**: Prisma (лучший DX для работы с типами и БД)
- **Auth**: Clerk (простая и надежная авторизация) или Auth.js
- **Maps**: React-Leaflet (бесплатные карты)
- **Forms**: React Hook Form + Zod

## Основной функционал (MVP)

1. **Каталог студий**:
   - Фильтрация (цена, площадь, оборудование, естественный свет).
   - Карта с метками.
2. **Страница студии**:
   - Галерея залов.
   - Календарь занятости.
   - Список оборудования.
3. **Личный кабинет**:
   - Для владельца: Добавление/редактирование студий и залов.
   - Для фотографа: История бронирований, Избранное.
